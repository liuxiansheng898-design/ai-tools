## 📖 工具简介

### AI提示词助手
- 支持 Midjourney、Stable Diffusion、DALL-E
- 6种风格：写实、动漫、电影、中国风、奇幻、科幻
- 一键生成提示词，复制即可使用

### AI视频提示词
- 专为可灵、即梦、Runway、Pika设计
- 多种运镜方式：慢动作、推拉、环绕、横移
- 电影级质量参数

### 短视频脚本
- 6种类型：干货、情感、搞笑、美食、种草、剧情
- 15/30/60秒时长可选
- 结构化脚本格式

### 小红书标题
- 爆款标题模板
- 批量生成
- 带话题标签

### 八字算命（娱乐版）
- 基础五行分析
- 性格特点
- 仅供娱乐参考

### AI取名
- 宝宝起名
- 国风/现代/诗意风格
- 男女通用

### 心灵鸡汤
- 每日正能量语录
- 一键分享

### 幸运数字
- 随机幸运数字
- 今日运势
