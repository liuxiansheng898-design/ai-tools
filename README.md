## 📖 使用教程

### AI提示词助手
1. 输入主体描述（如：一只可爱的猫）
2. 选择类型（文生图/文生视频）
3. 选择风格（写实/动漫/电影/中国风/奇幻/科幻）
4. 点击生成，一键复制提示词

### 短视频脚本
1. 输入视频主题
2. 选择视频类型（干货/情感/搞笑/美食/种草/剧情）
3. 选择时长（15秒/30秒/60秒）
4. 一键生成完整脚本
